|      符号      |        作用        |
| :------------: | :----------------: |
|       =        |        等于        |
|     <>、!=     | 不等于(不同版本可) |
|       >        |        大于        |
|       <        |        小于        |
|       >=       |      大于等于      |
|       <=       |      小于等于      |
|      \|\|      |      字符连接      |
|       %        |   匹配任意多字符   |
|       _        |   匹配任意一字符   |
|     [a-z]      |      匹配a-z       |
|     [0-9]      |      匹配0-9       |
| [^abc]、[!abc] | 不匹配任意单一字符 |
|  Regexp_Like   |      匹配正则      |
|      LIKE      |    搜索某种模式    |
|  BETWEEN AND   |    在某个范围内    |
|     EXISTS     |   返回结果集为真   |
